# 📚 CoursEz

CoursEz is a Learning Management System where students can learn online by enrolling in courses, tracking their progress, and rating them.  
Educators can create and publish their own courses and decide whether lectures are free or paid.

It helps solve the problem of limited access to quality education by making expert-created content available online.

---

## 🚀 Features

### 👨‍🎓 Student
- Sign up / login using Clerk authentication  
- Enroll in courses  
- View enrolled courses  
- Watch video lectures (streaming)  
- Mark lectures as completed  
- Track course progress  
- Rate courses  

### 👨‍🏫 Educator
- Create and publish courses  
- Add chapters and lectures  
- Set lectures as free or paid  
- View dashboard  
- See enrolled students  

---

Course → Chapters → Lectures (video URL)
